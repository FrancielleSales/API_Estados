## Endpoints

- **Post** -/api/states

  *Só aceita os seguintes valores:* Norte, Nordeste, Sul, Sudeste e Centro-Oeste.

- **Get** -/api/states

  *Possui as opções de filtrar os estados por região, maior população e maior área.* 

- **Get** -/api/states/{id}

- **Put** -/api/states/{id}

  *Só aceita os seguintes valores:* Norte, Nordeste, Sul, Sudeste e Centro-Oeste. 

- **Delete** -/api/states/{id}
